À propos de SatVault

Un projet né d'un hackathon, pensé pour un vrai problème.

En Afrique de l'Ouest, des millions de personnes n'ont pas accès à un compte bancaire fiable. Les transferts d'argent coûtent cher, prennent du temps, et l'épargne formelle reste hors de portée.

SatVault utilise le réseau Bitcoin Lightning pour offrir des paiements quasi-instantanés à coût quasi-nul, une épargne programmée et verrouillée dans le temps, et la possibilité de recevoir des fonds même sans connexion internet — grâce aux vouchers eCash Cashu.

Problèmes résolus

🏦
Pas de banque ? → Pas de problème
SatVault est un wallet Bitcoin auto-custodial. Aucun compte bancaire requis, aucun KYC contraignant.
💸
Frais de transfert élevés → Sats instantanés
Les paiements Lightning coûtent des fractions de centime, en moins d'une seconde, partout dans le monde.
📵
Pas de réseau → Vouchers eCash
Générez des tokens Cashu hors-ligne. Envoyez-les par SMS ou messagerie. Racheter quand le réseau revient.
🎯
Dépenses impulsives → Épargne verrouillée
Chaque objectif crée un wallet isolé, bloqué jusqu'à la date choisie. Vos sats sont inaccessibles avant l'échéance.

Fonctionnalités clés

Quatre piliers pour une liberté financière réelle.

⚡
Wallet Lightning Principal
Envoyez et recevez des sats instantanément. Support complet du protocole bolt11 et LNURL-pay. Solde affiché en sats, BTC et en devise locale (XOF / CFA). Notifications en temps réel via Supabase Realtime.
bolt11 LNURL QR Code Realtime
🏦
Épargne Programmée
Créez des objectifs d'épargne avec un montant cible et une date de déblocage. Chaque objectif possède son propre wallet LNbits isolé. Les fonds sont verrouillés côté serveur — impossible de retirer avant l'échéance.
LNbits Time-lock Wallet isolé
📵
Réception Hors-ligne
Générez des vouchers eCash Cashu depuis votre wallet Lightning. Partagez-les par n'importe quel canal (SMS, WhatsApp, papier). Le destinataire les rachète quand il retrouve du réseau.
Cashu eCash Offline
🔐
Sécurité Zero-Trust
Aucune clé secrète dans le client mobile. Les Admin Keys LNbits sont chiffrées AES-256 dans Supabase Vault. Toutes les opérations sensibles passent par des Edge Functions Supabase avec authentification JWT.
AES-256 Supabase Vault JWT Edge Functions

Flux d'authentification et de paiement

1
Magic Link L'utilisateur reçoit un lien Supabase Auth par email. Aucun mot de passe stocké.
2
JWT Supabase Token JWT émis, stocké de manière sécurisée dans SecureStore Expo.
3
Edge Function appelée Le client envoie le JWT + paramètres. La fonction vérifie l'identité.
4
Vault déchiffré La fonction récupère la clé LNbits AES-256 depuis Supabase Vault.
5
LNbits API appelée La transaction Lightning est exécutée. La clé ne quitte jamais le serveur.
6
Realtime push Supabase Realtime notifie le client du succès. Solde mis à jour.
